Detailed Description

template<class OrderedList, class Traits = michael_map::type_traits>
template<bool IsConst>
class cds::container::MichaelHashMap< gc::nogc, OrderedList, Traits >::iterator_type< IsConst >

Forward iterator.

IsConst - constness boolean flag

The forward iterator for Michael's map is based on OrderedList forward iterator and has some features:

  • it has no post-increment operator, only pre-increment
  • it iterates items in unordered fashion
