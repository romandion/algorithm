Detailed Description

template<typename... Options>
struct cds::container::split_list::make_traits< Options >

Metafunction converting option list to traits struct.

Available Options:

  • split_list::ordered_list - a tag for ordered list implementation. See split_list::ordered_list for possible values.
  • split_list::ordered_list_traits - type traits for ordered list implementation. For MichaelList use container::michael_list::type_traits, for LazyList use container::lazy_list::type_traits.
  • plus any option from intrusive::split_list::make_traits
