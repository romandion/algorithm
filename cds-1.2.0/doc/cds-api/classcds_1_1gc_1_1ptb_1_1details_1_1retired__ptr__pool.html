Detailed Description

template<class Alloc = CDS_DEFAULT_ALLOCATOR>
class cds::gc::ptb::details::retired_ptr_pool< Alloc >

Pool of retired pointers.

The class acts as an allocator of retired node. Retired pointers are linked in the lock-free list.
