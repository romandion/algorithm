Detailed Description

Thread's hazard pointer manager.

To use Hazard Pointer reclamation schema each thread object must be linked with the object of ThreadGC class that interacts with GarbageCollector global object. The linkage is performed by calling cds::threading::Manager::attachThread() on the start of each thread that uses HP GC. Before terminating the thread linked to HP GC it is necessary to call cds::threading::Manager::detachThread().
