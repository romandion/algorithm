Detailed Description

template<typename GC, typename T, typename... Options>
class cds::container::MoirQueue< GC, T, Options >

A variation of Michael & Scott's lock-free queue.

It is non-intrusive version of intrusive::MoirQueue.

T is a type stored in the queue. It should be default-constructible, copy-constructible, assignable type.

Options description see MSQueue

Clear the queue.

template<typename GC , typename T , typename... Options>
bool cds::container::MoirQueue< GC, T, Options >::dequeue ( value_type &  dest)
inline

Dequeues a value from the queue.

If queue is not empty, the function returns true, dest contains copy of dequeued value. The assignment operator for type value_type is invoked. If queue is empty, the function returns false, dest is unchanged.

template<typename GC , typename T , typename... Options>
bool cds::container::MoirQueue< GC, T, Options >::enqueue ( const value_type &  val)
inline

Enqueues val value into the queue.

The function makes queue node in dynamic memory calling copy constructor for val and then it calls intrusive::MSQueue::enqueue. Returns true if success, false otherwise.
