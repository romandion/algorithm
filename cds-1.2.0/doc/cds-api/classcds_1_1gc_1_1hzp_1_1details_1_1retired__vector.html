Detailed Description

Array of retired pointers.

The vector of retired pointer ready to delete.

The Hazard Pointer schema is build on thread-static arrays. For each HP-enabled thread the HP manager allocates array of retired pointers. The array belongs to the thread: owner thread writes to the array, other threads just read it.
