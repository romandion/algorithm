Detailed Description

template<class GC, typename Lock = lock::Spin, typename Tag = opt::none>
struct cds::intrusive::lazy_list::node< GC, Lock, Tag >

Lazy list node.

Template parameters:

  • GC - garbage collector
  • Lock - locm type. Default is cds::lock::Spin
  • Tag - a tag used to distinguish between different implementation
