Returns smallest following power of 2 for n.

Examples:

ceil2(0) == 1 // !!!
ceil2(1) == 1
ceil2(2) == 2
ceil2(3) == 4
ceil2(4) == 4
ceil2(15) == 16
ceil2(16) == 16
ceil2(17) == 32
