Detailed Description

template<typename T, typename TVOID, typename TNOVOID = T>
struct cds::details::void_selector< T, TVOID, TNOVOID >

Void type selector.

Returns in typedef type result of

if ( T == [possibly cv-qualified ] void ) type := TVOID
else type := TNOVOID

The TNOVOID is optional, default is TNOVOID == T
